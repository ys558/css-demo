## css3部分特性应用:

### 1. 选择器:
#### 1.1 后代选择器: 即使多层嵌套也行
语法: 空格

示例代码:
```css
  div p {}
```

### 1.2 子代选择器: 即使多层嵌套也行, 和1.1后代选择器类似, 但效率比后代选择器高
语法: >

示例代码:
```css
  div > p {}
``` 

### 1.3 紧邻选择器:
语法: +

示例代码: 
```css
  /* 紧挨着div的下面一个p标签 */
  div + p {}
```

### 1.4 紧邻之后的所有元素的选择器
语法: ~

[示例代码]('D:\study\code\css3-news-feature\01_css3的样式选择器.html): 
```css
  /* 紧挨着strong的所有a标签 */
  strong~a { }
```
